# TextReplacer

macOS에서 텍스트를 자동으로 치환해주는 메뉴바 앱입니다.

## 기능

- 실시간 키보드 입력 모니터링
- 트리거 문구를 감지하여 자동으로 텍스트 치환
- 메뉴바에서 치환 규칙 추가/수정/삭제
- 규칙을 JSON 파일로 자동 저장

## 사용 예시

트리거: `;wkaeupon`
치환: `sudo pmset disablesleep 1`

`;wkaeupon`을 입력하면 자동으로 `sudo pmset disablesleep 1`로 변환됩니다.

## 빌드 방법

```bash
cd TextReplacer
chmod +x build.sh
./build.sh
```

## 실행 방법

```bash
open build/TextReplacer.app
```

## 설치 방법

Applications 폴더에 복사:

```bash
cp -r build/TextReplacer.app /Applications/
```

그 다음 Applications 폴더에서 TextReplacer 앱을 실행하세요.

## 입력 모니터링 권한 설정 (중요!)

⚠️ **macOS 보안 정책으로 인해 수동으로 권한을 추가해야 합니다.**

앱을 처음 실행하면 권한 안내 메시지가 표시됩니다. 다음 단계를 따라주세요:

1. **시스템 설정** > **개인정보 보호 및 보안** > **입력 모니터링** 으로 이동
2. 왼쪽 하단의 🔒 **자물쇠를 클릭**하고 비밀번호 입력
3. 왼쪽 목록 하단의 **[+] 버튼 클릭**
4. **Applications 폴더**에서 **TextReplacer** 선택하고 **열기** 클릭
5. **TextReplacer** 옆의 **체크박스 활성화**
6. 앱을 재시작 (메뉴바에서 종료 후 다시 실행)

## 사용 방법

1. 앱이 실행되면 메뉴바에 ✏️ 아이콘이 나타납니다
2. 아이콘을 클릭하면 현재 설정된 치환 규칙을 볼 수 있습니다
3. **새 규칙 추가**를 클릭하여 새로운 치환 규칙을 만들 수 있습니다
4. 기존 규칙을 클릭하면 편집하거나 삭제할 수 있습니다
5. 앱을 비활성화하려면 메뉴에서 **비활성화**를 선택하세요 (아이콘이 ⏸️로 변경됩니다)

## 치환 규칙 관리

### 규칙 추가
1. 메뉴바 아이콘 클릭
2. **새 규칙 추가...** 선택
3. 트리거와 치환 텍스트 입력
4. **추가** 클릭

### 규칙 편집/삭제
1. 메뉴바 아이콘 클릭
2. 수정하려는 규칙 클릭
3. 치환 텍스트를 수정하거나 **삭제** 버튼 클릭

### 저장 위치
치환 규칙은 다음 위치에 자동 저장됩니다:
```
~/Library/Application Support/TextReplacer/rules.json
```

## 기본 규칙

앱을 처음 실행하면 다음 기본 규칙이 자동으로 추가됩니다:
- 트리거: `;wkaeupon`
- 치환: `sudo pmset disablesleep 1`

## 시스템 요구사항

- macOS 11.0 (Big Sur) 이상
- Xcode Command Line Tools (Swift 컴파일러)

## 문제 해결

### 텍스트 치환이 작동하지 않는 경우
- 입력 모니터링 권한이 제대로 설정되었는지 확인
- 앱을 재시작
- 메뉴바에서 앱이 활성화되어 있는지 확인 (✏️ 아이콘)
- 시스템을 재시동

### 빌드가 실패하는 경우
Xcode Command Line Tools 설치:
```bash
xcode-select --install
```

### 특정 앱에서 작동하지 않는 경우
일부 앱(예: 비밀번호 입력창)은 보안상의 이유로 입력 모니터링이 제한될 수 있습니다.

## 팁

- 트리거는 독특한 문자열을 사용하는 것이 좋습니다 (예: `;`, `:`, `//` 등으로 시작)
- 자주 사용하는 긴 명령어나 이메일 주소 등을 저장해두면 편리합니다
- 트리거가 일반 단어와 겹치지 않도록 주의하세요
